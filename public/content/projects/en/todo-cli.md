# ToDo CLI

## Description

A command-line tool for managing daily tasks. Allows users to add, view, complete, and delete tasks from a local JSON file.

## Features

-   Add new tasks
-   Mark tasks as complete
-   Delete completed tasks
-   List pending tasks

## Technologies

-   Python
-   JSON
-   argparse

## Future Improvements

-   Task priorities
-   Due dates
-   Sync with cloud
